# MENT 2234: The Entrepreneurial Journey

This repository contains materials for MENT 2234: The Entrepreneurial Journey — a course about nurturing the entrepreneurial mindset through systems, simulations, and practice.

---

## Course Philosophy

> "Chasing happiness but saved myself through nurturing the entrepreneurial mindset."

This course is built on the idea that entrepreneurship is not just about starting companies — it's about building systems that create value, improve lives, and generate financial freedom. Whether you're running a side hustle, a small business, or aiming for an IPO, the same fundamental mindset applies.

### Week 3: Entrepreneurial Mindset as System
Entrepreneurship is a large spectrum — side hustle, small business, medium business, IPO firms.
*   **[Entrepreneurial Mindset](./slides_mindset/MindsetSystems.html)** — It's about systems, not just passion

### Week 5: Use Entrepreneurial Mindset to Build Financial Freedom
What is FIRE? Different levels. The same process applies.
*   **[Financial Freedom](./slides_mindset/FinancialFreedom.html)** — Building entrepreneurial spirit as your asset
*   **Simulation:** Show how startups build their financial freedom
*   **Reflection:** Map your path to financial independence

## Module 2: Business Model Development (Weeks 6-10)

Practice your business mindset through a structured process:
1. **Identify the problem**
2. **Form hypotheses to provide solutions**
3. **Gather data to test the hypotheses and solutions**
4. **Revise your hypotheses and solution**

### Week 6: Problem Identification
