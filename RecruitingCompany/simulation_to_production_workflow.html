Intake Pipeline — The Only Net-New System

🎙️ Voice Calls
Whisper / Deepgram → transcript
📧 Email / Text
IMAP / webhook → parsed
📄 Documents
PDF, image, DOCX → OCR
🔗 API Webhooks
CRM, ERP, ticketing
→

Intake Pipeline

1. Connector ingests raw input
2. Extract text + files
3. Claude extracts structured fields
4. Normalizer → NormalizedTask
5. Chunk + embed into ChromaDB
→

Task Creator

Create group (case)
Store files as RunMaterials
Inject context into steps
Start orchestration engine
→
🤖 Orchestrator runs step DAG
👤 Workers claim HITL steps
💬 AI assistant provides context
📡 Signals record everything
